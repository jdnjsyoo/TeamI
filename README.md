# 🚇 지하철 자리전쟁

서울 지하철 2호선에서 자리를 차지하기 위한 치열한 경쟁을 체험하는 인터랙티브 웹 게임입니다.

## 📖 게임 소개

피곤한 하루를 보낸 당신, 지하철에서 자리에 앉고 싶지 않으신가요? 하지만 서울 2호선의 자리는 아무에게나 허락되지 않습니다! 이 게임은 여러분의 지하철 경험치를 시험하며, 3개의 라운드에서 좌석을 차지할 수 있는 기회가 주어집니다.

## 🎮 게임 방법

### Round 1: "이번 역에서 내릴 사람은 누구?"
- **목표**: 현재 역에서 내릴 것 같은 승객 찾기
- **조작법**: 
  - `Space Bar`: 라운드 시작
  - `←/→ 방향키`: 플레이어 이동
  - 승객 앞에 서서 대기 또는 `Space Bar` 클릭
- **성공 조건**: 제한 시간 내에 정답 NPC 앞에 서 있기
- **힌트**: 
  - 바깥 풍경에서 역 이름 확인하기
  - 승객들의 옷차림과 행동 관찰하기
  - 힌트 모션을 보이는 승객에 주목하기

### Round 2: "엇, 저기 자리 났다!"
- **목표**: 빈 자리까지 빠르게 달려가기
- **조작법**:
  - `Space Bar`: 라운드 시작
  - `→ 방향키`: 오른쪽으로 달리기 (한 번에 10px 이동)
  - `S키` : 누를 때마다 가속
  - 7번 자리 영역에서 `sit_arrow.png` 클릭
- **성공 조건**: 제한 시간 내에 7번 자리에 착석하기

### Round 3: "물러나라 지하철 빌!런!"
- **목표**: 3-4번 자리 사이에 서 있는 빌런을 밀어내고 자리 차지하기
- **조작법**:
  - `←/→ 방향키`: 3-4번 자리 중에서 정답 NPC 선택
  - `Enter`: 눈빛을 보내 빌런 밀어내기
- **성공 조건**: 빌런을 밀어내고 비워진 자리에 앉기 (해당 승객이 실제로 하차해야 함)

### 결과 화면
- Round 1, 2, 3에서 획득한 점수에 따라 최종 결과 화면이 나타납니다.
- `N` 키를 누르면 다음 라운드로 진행됩니다.

## 🎯 게임 특징

### 지하철역 시스템
게임에는 서울 지하철 2호선의 8개 역이 라운드 별로 랜덤 등장합니다:
- 강남, 강변, 서울대입구, 성수, 시청, 을지로, 잠실, 홍대

### NPC 시스템
- **총 7개의 자리**에 각각 다른 역을 대표하는 NPC가 랜덤 배치
- **정답 NPC (1명)**: 현재 역과 일치하는 승객
  - 일반 앉은 모션 + 힌트 모션을 반복
- **오답 NPC (6명)**: 다른 역의 승객들
  - 일반 앉은 모션만 반복

### 애니메이션
각 NPC는 다음과 같은 이미지로 구성됩니다:
- **sitting**: 앉아있는 모션 (2~3개 프레임)
- **hint**: 힌트 모션 (정답 NPC만 표시)
- **standing**: 서 있는 모션

## 🛠️ 기술 스택

- **p5.js**: 메인 게임 엔진 및 그래픽 렌더링
- **p5.sound.js**: 사운드 효과
- **JavaScript**: 게임 로직
- **HTML5/CSS3**: 웹 인터페이스

## 🚀 실행 방법

1. 프로젝트 클론 또는 다운로드
2. 로컬 웹 서버 실행 (예: Live Server 확장 프로그램 사용)
3. `index.html` 파일 열기
4. 게임 시작!
